# Sistema de Metas
Sistema desenvolvido para processar planilha de metas

## Funcionalidades

- Cadastro de Usuários, Supervisores.
- Importação da planilha de metas
- Alteração do valor das metas e aprovação da alteração

## Tecnologias

- Laravel.
